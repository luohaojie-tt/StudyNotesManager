## 🗂️ 文档说明

### 文档分类原则

- **00-overview**: 项目级概览文档
- **01-requirements**: 需求和功能定义
- **02-architecture**: 架构和技术设计
- **03-tasks**: 任务规划和进度追踪
- **04-backend**: 后端开发文档
- **05-frontend**: 前端开发文档
- **06-testing**: 测试相关文档
- **07-deployment**: 部署和运维文档
- **08-reports**: 完成报告和总结
- **09-workflow**: 工作流程和团队规范

### 文档维护规范

1. **更新文档时**：同步更新相关索引
2. **新增文档时**：添加到对应的分类目录
3. **过时文档**：移动到 `docs/archive/` 目录
