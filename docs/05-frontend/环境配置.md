## API Endpoints Used

- `GET /api/notes` - List notes
- `POST /api/notes` - Create note
- `POST /api/notes/upload` - Upload file
- `GET /api/mindmaps/note/:id` - Get mind map
- `POST /api/mindmaps/generate/:id` - Generate mind map
- `POST /api/quizzes/generate/:id` - Generate quiz
- `POST /api/quizzes/:id/submit` - Submit quiz
- `GET /api/mistakes` - List mistakes
- `PUT /api/mistakes/:id/review` - Mark reviewed
- `GET /api/analytics/stats` - Get statistics
- `GET /api/analytics/study-time` - Study time data
- `GET /api/analytics/accuracy` - Accuracy data
